# Projeto Av1 Parte 2 - Bianca Vitória

Este é um projeto simples de front-end contendo um arquivo HTML, CSS e JavaScript.

## Estrutura do projeto

- `index.html` - página principal
- `css/style.css` - estilos visuais
- `js/script.js` - lógica JavaScript

## Como usar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Edite `css/style.css` para modificar estilos.
3. Edite `js/script.js` para alterar comportamentos.

## Observações

- Não há dependências externas.
- Pode ser executado diretamente localmente (sem servidor).
